# 欢迎来到 CNC-CMD 知识共享平台

本平台采用MkDocs+GitHub Pages搭建，于2024年12月26日建立，欢迎大家加入。
旨在分享平时学习过程中遇到的好的技术文章、视频、项目等不限计算机相关的知识，帮助大家一起共同学习和进步。  
**该项目是帮助大家学习和入门的，虽然很基础，但是万一有些新人就恰恰需要呢？**  
注：如有侵权，请联系我们进行删除，谢谢。
